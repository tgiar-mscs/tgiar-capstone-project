WEATHER HELP
  • 1. If Outside, Seek Shelter Immediately
  • 2. If Indoors, Grab Supplies If Able, cans of food, water, flashlight
  • 3. The Shelter Should Be Away From Windows And Doors
  • 4. Once Weather Has Passed, Call For Help If Needed
  • 5. Listen For Others Whok May Need Help
